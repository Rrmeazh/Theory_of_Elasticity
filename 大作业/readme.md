# 弹性力学大作业

> 本研究由 Google Gemini 3 Pro 与 Claude Sonnet 4.5 鼎力支持

本研究是关于『球冠状薄壳受集中力作用时形变发生对称性破缺』的仿真。运行代码前请先安装 `code/requirements.txt` 中要求的库。

---

附：批改意见

本项工作相当优异，研究思路清晰、数值方法自己编程实现。

- (1)本研究正确取消了单一的初始条件限制，以保障了对称性破碎的发生
- (2)尽管在数值求解的角度上，通过有限差分格式实现的空间梯度会引入额外的阻尼，但鉴于
  - ①本系统求解的是稳态系统。
  - ②这些额外的阻尼提供了扰动力，从而实现了屈曲构形，尽管本文未详细进行探讨，但以后若有机会可以进行更进一步的研究。
- (3)但本研究也存在一定不足，为什么本研究探讨的是从 n=4 开始的模态演化，这一部分没有进行更详细的理论分析。
